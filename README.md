# Fullstack Concert Ticket

This monorepo hosts a Next.js frontend and a NestJS backend for the concert ticket platform.

## Frontend
- Located in `frontend`
- Powered by Next.js 16 with React 19 and TailwindCSS
- Common scripts: `npm run dev`, `npm run build`, `npm run start`

## Backend
- Located in `backend`
- Built with NestJS 11
- Common scripts: `npm run start:dev`, `npm run build`, `npm run test`

## Getting Started
1. Install dependencies for each project: `npm install` inside both `frontend` and `backend` directories.
2. Run the backend locally with `npm run start:dev` inside `backend`.
3. Run the frontend locally with `npm run dev` inside `frontend`.
