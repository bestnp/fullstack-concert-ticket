## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

### Environment

This API expects a PostgreSQL database configured through Prisma. Create a `.env` file at the project root with the following values:

```env
DATABASE_URL=postgresql://<user>:<password>@<host>:<port>/<database>
DIRECT_URL=postgresql://<user>:<password>@<host>:<port>/<database>
PORT=3001
CORS_ORIGIN=http://localhost:3000
```

- `DATABASE_URL` / `DIRECT_URL` should point to the same database (Supabase or local Postgres).
- Adjust `PORT` if you run the backend on another port.
- `CORS_ORIGIN` accepts a comma-separated list of origins allowed to call this API.

Run Prisma migrations before starting the server:

```bash
npm run prisma:migrate:dev
```

(Or `npx prisma migrate deploy` for production.)
